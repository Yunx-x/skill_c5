# 青云神通技能描述

## 雷光遁龙诀<玄>

**技能ID**: `待补充`

**最大等级**: 20

**技能类型**: 主动攻击

**攻击范围**: 单体攻击 16/21米

**施法时间**: 1秒

**技能冷却**: 4.5秒

**技能效果**:
- 攻击目标3次，每击附加本体攻击力12%/228%，附加694/3171点攻击力
- 附加自身真气上限10%的攻击力
- 攻击怪物时每击额外附加本体攻击力12%/228%，附加攻击力694/3446点
- 附加自身气血真气上限和40%的攻击力
- 技能达20级时，4.5/13.5秒内令目标受到伤害加深，增加目标受到的伤害5%

## 神剑御雷真诀<玄>

**技能ID**: `待补充`

**最大等级**: 20

**技能类型**: 群体攻击

**攻击范围**: 自身周围 25米，目标限制29/30个

**施法时间**: 6秒

**技能冷却**: 30秒

**特殊效果**: 真气贯穿（真气大于95%时追加自身真气上限15%的攻击力）

**技能效果**:
- 攻击自身周围25米内的敌人
- 追加本体攻击力21%/107%，每次附加攻击力865/2111点
- 在六秒内造成18次伤害
- 范围内敌对目标越多余伤越大
- 效果一：每一击均可令目标禁食3秒
- 效果二：最后一段攻击附加自身真气上限15%的攻击力
- 效果三：第1、3、5、7、9、11、13、15、17段令目标定身6秒，定身能力为自身定身抗性
- 效果四：第2、4、6、8、10、12、14、16、18段令玩家目标限定6秒，该能能法和该动度具为别降低21%
- 效果五：施法时额外附加致命一击率10%
- 效果六：范围内敌对目标越多伤害越额外增加施法者自身气血、真气和的2%与攻击力的20%
- 效果七：每段附加20%致命一击伤害
- 效果八：施法攻击令自身2秒内处于无敌状态
