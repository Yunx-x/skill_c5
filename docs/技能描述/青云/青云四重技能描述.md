# 青云四重技能描述

## 霜天雪舞

**技能ID**: `待补充`

**最大等级**: 9

**技能类型**: 群体攻击

**攻击范围**: 日周围 5米，目标限制6个

**施法时间**: 1秒

**技能冷却**: 60秒

**特殊效果**: 如成功减速，则2秒后追加气血伤害

**技能效果**:
- 攻击目标周围5米内的敌人1次，附加724/934点攻击力
- 有49%/59%几率令多目标减速25%，效果持续4秒
- 当自身攻击高于目标，可令目标周围冰冻2.5秒，令其陷入"霜冻"状态，使其按比例扣血，且多影响6个目标
- 霜燃之叠造成每秒为目标致郁的20%
